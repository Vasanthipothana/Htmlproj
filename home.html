What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages and applications. It tells the browser how to display content such as text, images, links, videos, and more.

History of HTML

HTML was invented by Tim Berners-Lee in 1991 while working at CERN. It was created to allow researchers to share and link documents over the internet using hyperlinks. Since then, HTML has gone through many versions:

  • HTML 1.0 (1993): The very first version
  • HTML 4.01 (1999): Introduced modern features like tables, forms
  • XHTML (2000s): A stricter version of HTML
  • HTML5 (2014): Modern version supporting audio, video, canvas, and responsive design

Basic HTML Tags

Tag Purpose Example
<h1> to <h6> Headings <h1>Title</h1>
<p> Paragraph <p>This is a paragraph</p>
<a> Link <a href="url">Link</a>
<img> Image <img src="img.jpg">
<ul> & <li> List <ul><li>Item</li></ul>

Basic Structure of an HTML Page

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>

HTML Elements

An element typically has an opening tag, content, and a closing tag:

<p>This is a paragraph</p>

HTML Attributes

Attributes give extra information about elements. Example:

<img src="image.jpg" alt="My Image">

HTML Comments

Comments are not shown in the browser:

<!-- This is a comment -->

HTML Lists

Unordered list:

<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

Ordered list:

<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>

Links and Images

<a href="https://example.com">Click Here</a>
<img src="banner.png" alt="Banner" width="300">

HTML Table

<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Vasanthi</td><td>22</td></tr>
</table>

Basic HTML Form

<form>
  <input type="text" placeholder="Your Name">
  <input type="submit" value="Submit">
</form>

Semantic Tags

Tags like <header>, <footer>, <article>, <nav>, and <section> describe the structure of a page more clearly.
